Shared workflow ## The Examples Reuse One Contract

1. Build X, t, and y

Start from observed covariates, treatment assignments, and outcomes.

2. Fit one estimator

Choose direct, weighting, or doubly robust estimators depending on the treatment type and estimand.

3. Build requested interventions

Create the treatment table that represents the treatment levels or treatment grid you want to evaluate.

4. Predict and compare

Interpret average potential outcomes, response curves, or benchmark error against known truth where available.

What changes across guides ## What Varies By Treatment Type

Binary and categorical guides

These guides focus on requested treatment levels and return average potential outcomes for those levels.

Continuous guides

These guides build a treatment grid and visualize the resulting dose-response curve.

Density-aware workflows

Continuous examples can plug treatment-density estimators into GPS-style estimators when the assignment model matters.

Synthetic benchmark pages

Benchmark examples add known ground truth so you can evaluate the shape and error of the estimated curve directly.
